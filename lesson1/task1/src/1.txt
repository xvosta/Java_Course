          примитивные типы:
            целочисленные типы:
            byte - 8 бит
            short - 16 бит. типа "короткий" int
            char - 16 бит. целое число, обозначающее символ utf-16
            int - 32 бита. целое число
            long - 64 бита. "увеличенный" int

            типы с плавающей точкой:
            float -32 бита, вещественное число
            double - 64 бита, вещественное число

            boolean - 32 бита, логический тип, принимающий значения true или false

            ссылочные типы:
            String - объект класса String (строка).
            Обертки примитивных типов Byte, Integer, Short, Character, Long, Double,
            Float, Boolean. Содержат методы для преобразования типов и всякое другое.
